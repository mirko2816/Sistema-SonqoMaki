# Módulos funcionales

Los módulos del monolito se incorporarán aquí a medida que se implemente cada capacidad aprobada.

Cada módulo puede agrupar sus casos de uso, reglas de dominio y adaptadores propios cuando aporten claridad. Las rutas, controladores HTTP, solicitudes y vistas continúan usando las convenciones habituales de Laravel. No se crearán capas, interfaces ni repositorios genéricos sin una necesidad concreta.

La siguiente iteración podrá crear el primer módulo, `Authentication`, sin obligar a generar por adelantado los módulos funcionales restantes.
